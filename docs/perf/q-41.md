# Q41

```sql
SELECT URLHash, EventDate, COUNT(*) AS PageViews FROM hits WHERE CounterID = 62 AND EventDate >= '2013-07-01' AND EventDate <= '2013-07-31' AND IsRefresh = 0 AND TraficSourceID IN (-1, 6) AND RefererHash = 3594120000172545465 GROUP BY URLHash, EventDate ORDER BY PageViews DESC LIMIT 10 OFFSET 100;
```

## Fast path

`group_fused_q41.rs` → `group_columnar::dashboard_q41_topk`:

1. PK zone index → dashboard date/counter ranges.
2. Parallel zone subranges; `for_each_q41_zone_match` (SIMD referer/traffic/refresh checks).
3. Collect packed `(URLHash, EventDate)` keys; `agg_sort::sorted_topk_u128`; top-K with OFFSET 100.

Fallback without zones: mask + sort on packed pair keys.
