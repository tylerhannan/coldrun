# Q36

```sql
SELECT ClientIP, ClientIP - 1, ClientIP - 2, ClientIP - 3, COUNT(*) AS c FROM hits GROUP BY ClientIP, ClientIP - 1, ClientIP - 2, ClientIP - 3 ORDER BY c DESC LIMIT 10;
```

## Fast path

`group_fused_q36.rs` → `group_columnar::clientip_quad_topk` — parallel copy ClientIP as u32, sort, run-length count, heap top-10; unpack quad for SELECT list.

No filter (`WHERE` none) — scans full 100M ClientIP column.

## 100M

Sort replaces sharded hash map (millions of unique IPs). Expect large drop vs old **~85s** smoke; ClickHouse **~0.27s**.

On 1M Parquet warm: **~0.13s** ([`compare-hot.md`](../benchmarks/parquet-hits-1m/compare-hot.md)).
